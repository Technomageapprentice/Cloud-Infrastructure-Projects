# Cloud-Infrastructure-Projects

AWS Cloud Infrastructure & Solutions Architecture
Welcome to my central hub for cloud architecture, infrastructure-as-code (IaC), and serverless experimentation. This repository serves as a live portfolio of my progress toward the AWS Certified Solutions Architect – Associate (SAA-C03) and beyond.

As a Sales Engineer, my goal is to design systems that aren't just technically sound, but are resilient, cost-optimized, and aligned with specific business outcomes.

🏗️ Active & Planned Projects
I am currently migrating and redacting production-grade automation tools for public view. Below is the roadmap for the "Proof of Concept" (PoC) builds currently in development:

1. Secure Global CAD Delivery System
The Business Case: Solving latency and security issues for global engineering firms sharing massive (GB+) CAD files.

Tech Stack: AWS S3, CloudFront (OAI/OAC), Lambda@Edge, IAM.

Status: In Design.

2. Linx: Browser-Based On-Screen Take-off
The Business Case: Replacing legacy, manual takeoff processes in the HVAC/Ductwork industry with a modern, high-speed web tool.

Tech Stack: Cognito, S3, Cloudfront, DynamoDB.

Status: Prototype Development.

3. LOS: Life Operating System (PKM)
The Business Case: A high-availability Personal Knowledge Management system built on a "Second Brain" methodology.

Tech Stack: S3 (Versioned), AWS Sync, Obsidian integration.

Status: Operational / Refinement.

4. Resilient Multi-Tier WordPress Architecture
The Business Case: A gold-standard deployment for high-traffic content management.

Tech Stack: EC2 (Auto-Scaling), RDS (Multi-AZ), EFS, Application Load Balancer (ALB).

Status: Planned.
